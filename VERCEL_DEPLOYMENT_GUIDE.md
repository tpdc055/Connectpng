## üöÄ Step-by-Step Vercel Deployment

### **Step 1: Go to Vercel**
1. **Visit**: https://vercel.com
2. **Sign up/Login** with your GitHub account
3. **You'll see the Vercel dashboard**

### **Step 2: Import Your Repository**
1. **Click "New Project"**
2. **Select "Import Git Repository"**
3. **Choose your GitHub repository** (the one with universal system)
4. **Click "Import"**

### **Step 3: Configure Project Settings**
**Framework Preset**: Next.js (should auto-detect)
**Root Directory**: `.` (leave default)
**Build Command**: `next build` (auto-filled)
**Output Directory**: `.next` (auto-filled)
**Install Command**: `bun install` or `npm install`

### **Step 4: Add Environment Variables**
**CRITICAL**: Click "Environment Variables" and add:

```bash
# Required Variables
DATABASE_URL=postgresql://your-database-url-here
JWT_SECRET=your-super-secure-jwt-secret-32-characters-long
SESSION_SECRET=your-session-secret-key

# Optional Variables
NEXT_PUBLIC_GOOGLE_MAPS_API_KEY=your-google-maps-api-key
NEXT_PUBLIC_APP_URL=https://your-project-name.vercel.app
```

### **Step 5: Set Up Database**
You need a PostgreSQL database. Choose one:

#### **Option A: Vercel Postgres (Recommended)**
1. **In Vercel Dashboard** ‚Üí **Storage** tab
2. **Create Database** ‚Üí **Postgres**
3. **Name**: `infrastructure-monitor-db`
4. **Region**: Choose closest to your users
5. **Copy DATABASE_URL** and add to environment variables

#### **Option B: External Database (Neon, Supabase, etc.)**
1. **Create PostgreSQL database** at Neon.tech or Supabase
2. **Copy connection string**
3. **Add to DATABASE_URL** environment variable

### **Step 6: Deploy**
1. **Click "Deploy"**
2. **Wait 2-3 minutes** for build and deployment
3. **You'll get a live URL**: `https://your-project-name.vercel.app`
